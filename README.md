- View a flipping visit card in a home page

<details>
  <summary>Run Locally</summary>

### Clone the project

```bash
  git clone https://github.com/Brice150/CARD.git
```

### Install dependencies

```bash
  npm install
```

### Start the server

```bash
  ng serve -o
```
